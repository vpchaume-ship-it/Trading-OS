# Order block

Dernière bougie opposée avant un déplacement franc ([[displacement]]) : la zone
où le « smart money » a chargé sa position. Support/résistance de qualité au
retour du prix. Cousin de l'[[ifvg|IFVG]] — souvent l'order block borde le gap
qui s'inverse. Confluence avec un [[sweep-de-liquidite|sweep]] de niveau de
session = setup A+ style [[../research/methode-dodgy|Dodgy]].

Non codé comme filtre distinct : le socle capte l'essentiel via IFVG + sweep +
[[v-shape|V-shape]]. Candidat d'[[ecole-des-presque|école des presque]] si une
cohorte le justifie un jour.
