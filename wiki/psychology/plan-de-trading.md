# Plan de trading

Le plan est écrit AVANT le marché, pas pendant. Pour ce système :
- **Seul setup autorisé** : [[../concepts/sweep-de-liquidite|sweep de session]] +
  [[../concepts/ifvg|IFVG]] + [[../concepts/v-shape|V-shape]], en
  [[../concepts/killzone|killzone NY AM]].
- **1 trade / jour** — la [[../lessons|discipline est un edge mesurable]].
- Risque figé AVANT l'entrée ([[gestion-du-risque]]), stop non négociable.
- Hors de ces conditions : **terminal fermé**. Ne pas trader = une décision
  gagnante la plupart du temps ([[biais-emotionnels]]).
